1. Clone the repository:
   ```bash
   git clone https://github.com/Soumyajit017/Striver-DSA-sheet.git
   ```

2. Navigate to any topic folder and run the Python files:
   ```bash
   cd Striver-DSA-sheet/Recursion
   python recursion_1.py
   ```
